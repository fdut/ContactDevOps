CONSIGNE DE SECURITE

1. La serrure est livrée avec un code de départ "3333". Nous vous recommandons de le remplacer tout de suite par votre code personnel comme décrit en point 2.

2. Notez votre code personnel et conservez-le dans un endroit inaccessible à d'autres personnes.

3. Cette serrure dispose d'un système brouilleur automatique qui remplace les molettes sur "0000" à chaque ouverture et fermeture.

Pour cette raison, "0000" ne doit être choisi comme code personnel.


OUVERTURE DE LA SERRURE

1. Pour ouvrir la serrure, faites "3333" ou votre code personnel "••••"

2. Tournez le bouton sur "OPEN"

3. Pour des raisons de sécurité, les molettes se déplacent automatiquement sur "0000"


CHANGEMENT DE CODE

1. En position ouverte, faites le code "3333" ou votre code personnel "••••"

2. Tournez le bouton sur "C"

3. Faites le nouveau code "NNNN"

4. Tournez le bouton sur "CLOSE"

5. Votre nouveau code est enregistré!


OUVERTURE D'URGENCE / DECODAGE

1. En position fermée, introduisez la clé et entournez vers la gauche jusqu'à la butée

2. Tournez le bouton sur "OPEN"

3. Les molettes se placent automatiquement sur le code actuel


GENERALITES

1. Protégez la serrure de l'eau, du feu, de la poussière

2. Installez la serrure dans des milieux compris entre 0°C et 50°C

3. Pour le nettoyage, prenez un chiffon humide. Ne pas utiliser de produits chimiques

4. Veillez à ce que les objets à l'intérieur de l'armoire ne gênent pas les éléments de fermeture

5. Ne pas graisser ou huiler la serrure ou les molettes
