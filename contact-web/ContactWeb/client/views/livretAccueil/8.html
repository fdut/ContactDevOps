De base dans une salle de réunion, le poste de réunion audio permet des appels internes, et externes nationaux.Pour appeler un correspondant international, il faut se connecter sur le poste avec son compte personnel. ( UID, PIN code téléphonique)


SE CONNECTER

1. Utiliser la touche dynamique "Apps"

2. Sélectionner « Extension Mobility » avec le navigateur et valider avec la touche de validation

3. Renseigner votre UID à l’aide des lettres sur le clavier Numérique puis votre PIN code

4. Valider avec la touche dynamique « Soum. ». Ne pas oublier de se déconnecter en partant (même manœuvre)


LE BOITIER DE COMMANDE

APPELER

1. Utiliser le clavier pour composer le numéro à 10 chiffres interne ou externe (avec le 0 devant ou 000 pour l’international)

2. Puis lancer l’appel avec la touche


REPONDRE A UN APPEL

Pour répondre à un appel, décrocher avec la touche

Ou utiliser la touche dynamique "Répondre"


RACCROCHER

Pour raccrocher utiliser à nouveau la touche d’appel
